About console

Now we have learned how to set Visual studio Code. Do you know about Console Screen? Well it is a screen which is mainly used to log information associated with a web page like: network requests, JavaScript, security errors, warnings, CSS etc. It enables us to interact with a web page by executing JavaScript expression in the contents of the page. For example create a var and name it as 'first name' and store it as 'Emma'. Then in the next line type 'console.log' and create tworound brackets and in it write variable name.
